☁️ Cloud Labs Portfolio
Hands‑on cloud security and infrastructure labs completed across AWS, Google Cloud Platform (GCP), and Microsoft Azure.
These labs demonstrate practical experience with identity management, virtual machines, web servers, image creation, and secure deployment workflows.

🌩️ AWS Lab — IAM Users, Groups & Policies
This lab focuses on building a secure Identity and Access Management (IAM) structure in AWS.
What I Did
- Created IAM users, groups, and roles
- Assigned least‑privilege permissions using AWS managed and custom policies
- Configured MFA for user accounts
- Tested access by logging in as different IAM users
- Explored how IAM integrates with EC2, S3, and other AWS services
Skills Demonstrated
- Identity & Access Management
- Principle of Least Privilege
- Policy creation and permission boundaries
- Multi‑Factor Authentication setup

🌐 Google Cloud Labs — VM Instances & Web Server Deployment
A series of GCP labs focused on creating and managing virtual machines, networking, and simple web server deployments.
What I Did
- Created Compute Engine VM instances (Linux & Windows)
- Configured firewall rules to allow HTTP/HTTPS traffic
- Installed and hosted a basic web server (Apache or Nginx)
- Managed VM snapshots, machine types, and disk settings
- Explored load balancing and external IP configuration
Skills Demonstrated
- Compute Engine VM provisioning
- Firewall & networking configuration
- Basic web server setup
- Instance management & snapshots

🔷 Azure Lab — Creating & Deploying a Generalized VM Image
This lab involved creating a custom virtual machine image, generalizing it, and deploying a new VM from that image.
What I Did
- Created a Windows or Linux VM in Azure
- Installed required tools, updates, and configurations
- Ran Sysprep (Windows) or generalized the VM (Linux) to prepare it as an image
- Captured the VM as a generalized image
- Deployed a new VM using the custom image
- Validated that the new VM booted correctly and inherited the base configuration
Skills Demonstrated
- Azure VM provisioning
- Image creation & generalization
- Understanding of Sysprep / VM preparation
- Deploying VMs from custom images
- Resource Group & Storage management
